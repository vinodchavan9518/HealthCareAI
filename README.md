# HealthCareAI

## Project Description
HealthCareAI is an AI-based healthcare management system developed using Java.

## Features
- Patient Registration
- Doctor Management
- Appointment Booking
- AI Health Suggestions
- Secure Login System

## Technologies Used
- Java
- JSP
- Servlet
- JDBC
- Oracle/MySQL Database
- HTML
- CSS

## How to Run
1. Open project in VS Code or Eclipse
2. Configure database
3. Run server
4. Open browser
